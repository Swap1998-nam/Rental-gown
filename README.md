# Rental Gown 👗

A modern and responsive gown rental web application built to help users browse, explore, and rent stylish gowns online with ease.

## 🌟 Features

* 👗 Browse different gown collections
* 🔍 Search and filter gowns
* 📱 Fully responsive design
* 🛒 Easy rental booking interface
* 💳 Modern UI/UX
* ⚡ Fast and lightweight website

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Version Control:** Git & GitHub

## 📂 Project Structure

```bash
Rental-gown/
│── index.html
│── style.css
└── README.md
```

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Swap1998-nam/Rental-gown.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd Rental-gown
```

### 3️⃣ Open in Browser

Simply open `index.html` in your browser.
